# Hotel Booking System - Database Schema

This project contains the SQL schema and ER diagram for a Hotel Booking System.

## Entities
- Guests
- Rooms
- Bookings
- Payments

## Relationships
- One Guest → Many Bookings
- One Room → Many Bookings
- One Booking → One Payment

- schema.sql (SQL table creation)
- er-diagram.png (ER diagram screenshot)
